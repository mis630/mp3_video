# Installation Guide - Smart Video & MP3 Downloader

## Quick Start (5 minutes)

### Step 1: Load the Extension in Chrome

1. Open Google Chrome
2. Navigate to `chrome://extensions/`
3. Enable **Developer mode** using the toggle in the top-right corner
4. Click **Load unpacked**
5. Select the folder containing this extension (`/workspace`)
6. The extension icon should appear in your toolbar!

### Step 2: Pin the Extension (Optional)

1. Click the puzzle piece icon (Extensions) in Chrome toolbar
2. Find "Smart Video & MP3 Downloader"
3. Click the pin icon to keep it visible

### Step 3: Start Using!

1. Click the extension icon
2. Paste any video URL from supported platforms
3. Click "Fetch Video"
4. Choose quality and download!

## Troubleshooting

### Extension not appearing?
- Make sure Developer mode is enabled
- Try refreshing the extensions page
- Check if the folder path is correct

### Can't load unpacked?
- Ensure you're in Chrome (not Edge or other Chromium browsers)
- Make sure the manifest.json file is in the selected folder
- Check that all files are present

### Icons not showing?
- The SVG icons should work in modern Chrome versions
- If needed, convert SVG to PNG using an online tool

### Permission errors?
- The extension needs clipboard, storage, and download permissions
- These are requested automatically when loading

## Updating the Extension

To update after making changes:
1. Go to `chrome://extensions/`
2. Find the extension
3. Click the refresh/reload icon
4. The changes will apply immediately
